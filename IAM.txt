Question: You have created a new compartment "apps" to host some production apps and you have created an apps_group and added users to it.
What would you do ensure the users access to the apps compartment?
Answer: Add an IAM Policy for apps_group granting access to the apps compartment

Question: You want to make API calls against other OCI Services from your instance without configuring user credentials.
How would you achieve this?
Answer: Create Dynamic Group and add a policy
