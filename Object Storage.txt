Question: At which level is versioning defined in object storage?
Answer: Defined at the bucket level

Question: At which level is replication set for object storage?
Answer: Bucket level

Question: You want to move some unstructured data, consisting of images and videos, to cloud storage.
Which is the most cost-effective storage that you would use?
Answer: Standard Storage

Question: You want to store the backup of a database in cloud storage for an extended period of time.
Which type of storage would you configure for these files?
Answer: Archive Storage

Question: You have created a bucket of ARCHIVE storage type. Which features and restrictions are applicable on that? (Choose two)
Answer: 
The default time to download an object after restoration is 24 hours
The minimum retention period for Archive storage is 90 days
